233版V2ray一键安装脚本 集成BBR/锐速/Shadowsocks

2019-02-23 分类：备胎专区 评论(0)

 

 

目前较为主流的科学上网代理工具中，除了用户众多的Shadowsocks/ShadowsocksR外，V2ray也有广泛的使用人群。

凭借强大的功能、不错的使用效果，V2ray正越来越被大家接受和认可，是Shadowsocks/SSR的重要备选工具。

之前，我们介绍过：V2Ray一键安装脚本 自带图形化界面控制面板，该版本自带网页控制面板，使用起来非常方便，但是也有些许不足：

  • 在部分系统环境下，V2ray可能会安装失败。
  • 暂时没有提供卸载功能。

鉴于以上两点，今天我们再介绍另外一款V2ray一键安装脚本，即网上流传也很广的233版本。两款脚本大家可以任意选择，以提高V2ray的安装成功率。

 

233版V2ray一键安装脚本  特点

优点：

  • 功能更为完善，安装成功率较高。
  • 脚本自带多项管理功能，后期维护方便。
  • 集成BBR/锐速一键安装（可选功能）。
  • 集成Shadowsocks一键安装，可与V2ray共存。
  • 安装完成后，可通过脚本快速修改V2ray/Shadowsocks各项参数。
  • 支持众多V2ray传输协议。
  • 支持广告屏蔽功能（可选）。
  • 支持Websocket+TLS/HTTP/2。
  • 支持动态端口。

不足：

  • 暂不提供网页控制面板功能。

 

233版V2ray安装环境要求

国外VPS服务器：

如果你手中还没有VPS服务器，可以参考：适合搭建代理的国外VPS服务器推荐汇总

系统要求：

CentOS 7+、Debian 8+、Ubuntu 16+，作者更推荐Debian 9.

时间要求：

VPS服务器时间 与 V2ray客户端所在设备时间，两者最好保持一致，不要误差太大。这是V2ray自身要求，与脚本无关。
putty连接服务器，输入date命令，可以查看服务器时间。

环境要求：

要求系统中装有curl，建议提前安装，安装命令：

yum update -y && yum install curl -y
 
# Debian/Ubuntu：
apt-get update -y && apt-get install curl -y

 

 

233版V2ray一键安装 流程

1.Putty连接VPS服务器，复制以下命令运行：

1
bash <(curl -s -L https://git.io/v2ray.sh)

 

2.在出现的菜单选择界面，输入数字1回车，开始安装。

 

3.下面会让你选择V2ray的传输协议，可选协议众多，如果没什么特殊需求的话，默认TCP即可。如果你需要启用类似KCPTUN的网络加速功能，则选择mKCP开头的几个，在mKCP协议下，流量以UDP的形式传输。

 

 

4.接下来分别需要选择：

  • V2ray端口：默认会给出一个端口，回车接受即可，或者也可以自定义。
  • 广告拦截：一般不需要开启，大家可能更习惯浏览器自带的广告拦截。
  • 是否配置Shadowsocks：如果需要同时安装Shadowsocks的话，选Y回车。建议安装，因为在V2ray出现无法连接的情况下，可以连接Shadowsocks用来判断是否服务器问题。

 

 

5.如果选择安装Shadowsocks，那么会依次要求设置端口、密码、加密协议，按提示设置即可。设置完成后回车键继续：

 

 

6.接下来会提示配置完成，可以开始安装，这时按回车键继续：

 

 

7.安装完成后，会出现下图界面，提示刚才的各项配置信息，可以截图保存，以备后续客户端连接使用：

提示：各项参数可以鼠标选择，选择成功后即自动复制。

 

 

8.现在可以使用V2ray客户端连接服务器了，连接成功后，就可以开始科学上网啦。

如何安装BBR/锐速

如果你的服务器系统，还没有安装BBR/锐速，那么可以利用脚本自带功能进行安装，成功安装后，可以为网络加速提高使用体验。

具体方法：

1.Putty连接VPS服务器，输入以下命令运行：

1
v2ray bbr

 

2.出现下图界面时，输入数字1安装BBR，或者输入数字2安装锐速：

注意：BBR和锐速不建议同时安装，二选一即可。
