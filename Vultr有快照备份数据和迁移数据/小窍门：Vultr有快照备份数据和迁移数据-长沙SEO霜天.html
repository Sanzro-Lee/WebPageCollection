小窍门：Vultr有快照备份数据和迁移数据

霜天 • 2019年3月11日 pm12:11 • 技术分享 • 阅读 222

Vultr提供免费的快照(Snapshot)功能，用户可以在管理面板轻松设置当前时间点的Snapshot且无需关闭主机。相比DigitalOcean每次设置Snapshot需要关机来说方便多了。使用国外服务器有时会遇到IP地址被GFW屏蔽，网站在国内无法访问。或者你使用的IP地址以前发布过大量的垃圾信息、虚假信息或者SEO作弊，被搜索引擎拉入黑名单，此IP下的站点迟迟不被收录，这时你就需要更换IP地址了。

Vultr VPS目前没有IP地址保留机制，也就是你每次新申请VPS的IP地址都有可能与上次不一样。因此你要更换Vultr VPS的IP地址也就简单多了。大概的步骤如下：

不定期更新，有事情在群里找我：QQ群：613294160

1、先使用Vultr VPS的免费快照功能，备份原服务器全部内容。

2、在Deploy页面使用快照新建一个VPS，Vultr 会为新的VPS分配一个全的IP地址。如下图，选择你刚才备份的快照。

接下来，就是备份你的服务器的密码，因为是同步过去的，所以，你的服务器密码也会被同步过去。

 

ok，当我们已经备份完成之后，我们就可以开始新建服务器了，跟着我的步骤走！

 

是不是很多人在测试这个服务器的账号密码，好像被你们发现了点什么。

然后测试的时候，始终进不去对不对，因为早就删了。哈哈

 

接下来教大家如何修改服务器密码:

[root@www ~]# passwd root   #输入代码
Changing password for user root.
New UNIX password: 
BAD PASSWORD: it is based on a dictionary word
Retype new UNIX password: # 重新输入
passwd: all authentication tokens updated successfully. #修改成功

如果你有一个已经分配了IP地址的VPS，也想继续使用这个IP地址，那你只需要在当前VPS的管理页面，选择快照恢复功能（找到相应的快照，点击Restore），恢复原来VPS的内容。
